# Nerve Repair Stack — Evidence Dashboard

An interactive, sortable research dashboard for **radicular lower back pain** (sciatica / lumbar disc herniation with nerve involvement). Covers every compound in the three-tier nerve-repair stack with published trial models, cytokines inhibited, pathways modulated, pain reduction metrics, and exact human dose protocols.

## 👉 Live Site

**[https://Sigward-Waters.github.io/pain-nerve-repair-dashboard/](https://Sigward-Waters.github.io/pain-nerve-repair-dashboard/)**

## Tier Structure

| Tier | Focus | Compounds |
|------|-------|-----------|
| 🟢 Tier 1 | Nerve Repair (axon/myelin regeneration) | ALCAR, ALA, Methylcobalamin B12, Vitamin D3, Lion’s Mane, Ashwagandha |
| 🔵 Tier 2 | Nerve Protection (anti-excitotoxic, anti-apoptotic) | PEA, Magnesium Glycinate, Hesperidin, Bacopa |
| 🟠 Tier 3 | Inflammation Suppression | Boswellia (AKBA), Curcumin+Piperine, Omega-3 EPA/DHA, Skullcap Baicalin, White Peony |

## Filters

- **Tier** — Repair / Protection / Inflammation
- **Evidence Grade** — Well-supported / Suggestive
- **Pain Type** — Neuropathic, Radiculopathy, Sciatica, Central Sensitization, Disc Degeneration, Muscle Spasm
- **Trial Type** — Human RCT / Animal / In vitro
- **Search** — Any cytokine or pathway (e.g. `IL-1β`, `NF-κB`, `NLRP3`, `NMDA`, `5-LOX`)

## Enable GitHub Pages

1. Go to **Settings → Pages**
2. Source: **Deploy from a branch → main → / (root)**
3. Save — live in ~60 seconds

## For Physician Review

Filter to **Evidence Grade: Well-supported** + **Trial Type: Human RCT** to see only the highest-quality evidence. Every dose column cites the protocol from the published trial, not the supplement label.
